ABOUT TRICK BOX...

This web application has been created in order to showcase different HTML5, CSS3 and JavaScript features. You can find some of them on the home page and other can be seen selecting different projects from the navigation bar. This project can bee seen on github. The status is work in progress as projects are going to be added continuously.


Some of the features presented by trick box:

  • single page application
  • app routing with location hashes
  • .svg file animation with JavaScript
  • navigation bar
  • scroll bar
  • favicon
  • API calls
  • dark mode / light mode
  • background picker
  • More features to be added soon...
